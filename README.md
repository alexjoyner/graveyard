CET-DATABASE
===============

## Install
    npm install
    npm start
    
## Testing
##### Main test scripts
    npm test
    or
    npm test:watch
    
##### Coverage Support
    npm run cover

## Docs

Currently, the absolute best documentation for this starter package is going to be in the tests of all the modules.  This project strives to be as close to 100% coverage as possible at all times.

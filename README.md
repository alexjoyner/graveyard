ro-utils
==========

**_v0.0.1 Currently just getting set up_**

A library set of quick utilities for all my projects.

## Installation
    `npm install ro-utils`


## Usages 
##### Add Two Numbers
    var roUtils = require('@alexjoyner/ro-utils');
    
    var sum = roUtils.addTwoNumbers(5, 6);
    
    Sum should equal 11;
